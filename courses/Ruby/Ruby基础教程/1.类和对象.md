##二、类的定义

###1.定义一个类

```
$ vim test.rb
```

输入代码：

```
 #!/usr/bin/ruby -w
 
 class Customer
     @@no_of_customer=0
     def hello(id, name, addr)
         @cust_id=id
         @cust_name=name
         @cust_addr=addr
     end
 end
```


* 以 `@@` 开始的是类变量

* 以 `@` 开始的是实例变量

* 以 `$` 开始的是全局变量
* 以小写或者是 `_` 开头的是局部变量


##三、创建类的对象

###1.对象

类中定义的方法称为类的成员函数。使用 new 关键字创建类的对象。

```
$ vim test.rb
```

输入代码：

```
 #!/usr/bin/ruby -w
 
 class Customer
     def hello
         puts &#34;Hello, World.&#34;
     end
 end
 
 cust1 = Customer.new
 cust2 = Customer. new # new 前面也可以有一个空格
 cust1.hello
 cust2.hello
```

运行代码：

```
$ ruby test.rb
```

###2.initialize 函数
initialize 函数是类初始化函数，创建对象时，会自动调用，并可通过 new 传递参数给 initialize 函数。

```
$ vim test.rb
```

输入代码：

```
 #!/usr/bin/ruby -w
 
 class Customer
     @@no_of_customer=0
     def initialize(name)
         @cust_name=name
     end
 
     def hello
         puts &#34;Hello, &#34; + @cust_name
     end
 end
 
 cust = Customer.new(&#34;Jack&#34;)
 cust.hello
```

运行代码：

```
$ ruby test.rb
```
